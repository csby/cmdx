# cmdx
command line tool

## command
```
Usage: 
    cmdx <command> [arguments] 
The command are: 
    help     show the command list
    folder   create, delete, clear, copy folder
```

### argument: folder
```
Usage: 
    cmdx folder <command> [arguments] 
The command are: 
    help     show the command list
    create   create folder, <target=folder path>
    delete   delete folder, <target=folder path>
    clear    clear folder, <target=folder path>
    copy     copy folder, <source=folder path> <target=folder path> [ignore=folder1|file1.txt|.git]
```
